Our lab has partnered with the start up company Improbable Beef to discover which bacteria will provide the perfect taste for a plant based meat. Our hypothesis is that the best bacteria for this project is found inside the human bellybutton. We have collected samples across the United States. All participants were annonymous and instead given an ID number. Below is the data collected on an interactive website to visualize all the data available at this time.
